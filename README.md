# kancolle-replay
Sortie Replay and Simulation by @fourinone41


---


## 概要
上記のforkで、シミュレートの実処理部分に日本語のコメントで処理の流れと型情報を補足したブランチです。処理自体の改変は原則として行いませんが、見やすくする為の改行等は若干行っています。

---

## インスタンス

### `kcships.js`

このファイルには以下のようなオブジェクトが含まれます：

1. **Fleet**
2. **Ship**
3. **Shipを継承した艦種(e.g. DD CV etc...)**
4. **Equip**
5. **LandBase**

これらはクラス (`class`) ではありませんが、役割としてはインスタンスに近いのでコメントではインスタンスとして扱っています。

### 注意

これらのオブジェクトでは、多くのプロパティが動的に設定されています。それらのプロパティはリファレンスが表示されません。オブジェクトの宣言部で初期値を設定すればリファレンスが通るようになりますが、処理に影響を与える可能性があるため、行っていません。エディタの画面分割などでコメント部を確認してもらうのが良いかと思います。
